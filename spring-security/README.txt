此项目包括如下功能。

1、Spring Security
2、Springload
3、EnableScheduling (定时器)
4、EnableAsync  (异步)
5、EnableCaching （对数据库操作缓存）
6、AOP统一处理Web请求日志
7、配置文件的自定义属性、随机数
8、RESTful 风格
9、Freemarker模板引擎渲染web视图
10、统一异常处理
11、使用Swagger2构建RESTful API
12、使用Spring-data-jpa
13、默认日志的配置

14、自定义 @RequestLimit(count=1, time = 2000)      	//限制每秒最多请求数
14、自定义 @Acl(chmod="R,R,R", chown="ison:message") //控制器级别和方法级别ACL鉴权控制
